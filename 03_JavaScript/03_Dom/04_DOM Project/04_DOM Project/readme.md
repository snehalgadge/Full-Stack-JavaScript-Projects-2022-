### i)Changing the background color

## Soltuion
``` 
let clearfix = document.querySelectorAll(".clearfix")
let arr = ["#EC9B3B", "#EE5487", "#F6901A", "#82BB30", "#4FACFF"]

for(let i=0; i<5; i++){
    clearfix[i].style.background = arr[i];
}
```
